 AI Study Companion

An AI-powered web app using Python, Streamlit, and Gemini API for personalized study planning, note summarization, and motivational support.

 Features
- Dynamic study planner
- Flashcard generator
- Sentiment-based motivational tips
- Note summarization

 Tech Stack
- Python
- Streamlit
- Google Gemini API
- OpenAI API (optional)
